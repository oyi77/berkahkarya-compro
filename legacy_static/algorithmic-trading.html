FAQ

Pertanyaan yang Sering Ditanyakan

Apakah sinyalnya auto-execute atau manual?

Plan Starter dan Pro menggunakan sinyal manual — kamu yang eksekusi di broker sendiri. Plan Elite mendapatkan akses API untuk integrasi auto-eksekusi dengan broker tertentu.

Broker apa yang kompatibel?

Sinyal bisa dieksekusi di broker MT4/MT5 mana pun. Untuk auto-eksekusi (Elite), saat ini mendukung broker dengan API terbuka. Tim kami akan bantu setup pada onboarding.

Win rate-nya berapa?

Backtesting strategi 7-Candle Breakout menunjukkan win rate 58-65% dengan risk:reward 1:2. Data live trading tersedia untuk diaudit. Ingat: tidak ada strategi dengan win rate 100%.

Modal minimum berapa untuk mulai?

Tidak ada minimum dari kami. Tapi kami rekomendasikan modal minimal $500 agar risk management bisa dijalankan dengan proper (max 1-2% per trade).

Apakah ada masa trial sebelum berlangganan?

Ada masa konsultasi gratis di awal. Hubungi tim kami via WhatsApp untuk demo sinyal selama 3 hari sebelum memutuskan berlangganan.
